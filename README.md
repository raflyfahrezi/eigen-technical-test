# EIGEN Technical Test - Farhan Rafly Fahrezi Saepulloh

## Stack

- Vite
- ReactJS
- EmotionCSS
- ANT Design

## How to run

### 1. Install the dependencies

```
yarn
```

### 2. Run the development server

```
yarn run dev
```

### 3. Open Browser

```
http://localhost:5173/
```

## How to Test

```
yarn run test
```
